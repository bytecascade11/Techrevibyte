Pehtheme is a fast, responsive, and beautiful Hugo theme featuring:
- Horizontal top navigation
- Featured posts on homepage
- Sidebar with recent posts and tags
- Built with Tailwind CSS (no extra JS)
- Fully compatible with Hugo extended

Start writing your content in the `content/posts/` folder!

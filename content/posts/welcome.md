Pehtheme is a clean, fast, and minimalist Hugo theme built with **Tailwind CSS**.  
No JavaScript bloat, fully responsive, and perfect for blogs or personal sites.

### Features
- Horizontal top navigation
- Featured posts on homepage
- Sidebar with recent posts and tags
- Pagination (6 posts per page)
- Dark mode ready
- Built with Hugo Extended
